# Draculinho Theme

Just another Dracula theme fork, optimized for ligatures and italic fonts.

Some colors modified to darker feelings, with my taste.

## Install

| Editor | Link |
|--------|------|
| VS Code | [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=eduardoborges.draculinho) |
| Cursor | [Open VSX Registry](https://open-vsx.org/extension/eduardoborges/draculinho) |
| VSCodium | [Open VSX Registry](https://open-vsx.org/extension/eduardoborges/draculinho) |

Or search for **Draculinho** in your editor's extension panel.
